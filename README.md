# GUI-Stop-Watch-
A fully functional desktop GUI Stopwatch built with Java and Swing. Developed as a 1-week sprint to practice core object-oriented programming (OOP), constructor chaining, and event-driven architecture using javax.swing.Timer and ActionListeners.
